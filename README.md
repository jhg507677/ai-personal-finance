# 가계부 애플리케이션 (Web & Mobile)

## 1. 프로젝트 개요

### 1.1 프로젝트 목적
- 클로브코딩을 활용하여 포트폴리오용 수익형 프로젝트 개발
- 광고 수익으로 서버 비용 충당 계획
- 클로브코딩 환경에 익숙해져 현업에서도 활용 가능하도록 설계
- 기존 무료 가계부 앱의 불편함 개선

### 1.2 도메인: 가계부
주요 기능:
- 간편한 수입/지출 기록 및 관리
- 일/월/년 단위 재무 현황 파악 (AI 기반 재무 조언 제공)
- 반응형 웹 디자인으로 모바일 최적화
- 직관적이고 사용하기 쉬운 UI/UX
- 가족/연인 간 거래 내역 공유 기능

### 1.3 대상 사용자
- 개인 재무를 관리하고자 하는 일반 사용자
- 간편하게 지출 내역을 기록하고 싶은 직장인
- 재무 계획을 세우고 실천하고자 하는 성인

---

## 2. 기술 스택

### 2.1 프론트엔드
- **프레임워크:** Next.js 14+ (App Router 방식)
- **언어:** TypeScript
- **스타일링:** Tailwind CSS
- **상태관리:** Redux
- **HTTP 클라이언트:** Axios

### 2.2 백엔드
- **프레임워크:** Spring Boot 3.0+
- **언어:** Java 17+
- **데이터 접근:** Spring Data JPA
- **데이터베이스:** MySQL 8.0+
- **빌드 도구:** Gradle
- **API 문서화:** Swagger
